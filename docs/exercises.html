Interactive Git Training Lab

Download from Twit Init Repository

00. Setup & Config

Install Git and configure your identity (name and email).

01. Init, Add, Commit

Learn how to track files and create your first snapshot (A:).

02. Branching Basics

Create parallel universes (B:) to work on features without affecting main.

03. Merge Conflicts

Resolve the clash (C: vs D:) when two branches fight over the same line.

04. Rebase vs Merge

Rewrite history (E: onto F:) to maintain a clean, linear project log.

05. Stash & Cleanup

Temporarily shelf your messy work in the waiting room.

06. Reset & Revert

Time travel to fix mistakes by undoing bad commits (H:).

07. Remote Workflow

Simulate a server by pushing your code to a local bare repository.

08. Pull Requests

Contribute to the real world by forking DevelSSN/twit and making a PR.

09. Tags & Releases

Mark your milestones with version tags (v1.0.0).

10. Cherry Pick

Selectively harvest specific great commits (I:) from other branches.

Usage: Navigate to each folder and run ./setup.sh to begin.
